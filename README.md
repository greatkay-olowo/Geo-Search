# Geo-Search

Geo-Search is a tool that allows people to search for places and extract information such as timezone, map and local weather conditions like temperature, wind speed, humidity, and precipitation.


## Getting Started

### Homepage

https://greatkay-olowo.github.io/Geo-Search/


### How to use

Input the city and country in the input fields. The api returns the weather conditions and the map of the city as well as a button to share the information on facebook.


### clone

https://github.com/greatkay-olowo/Geo-Search.git


## Built With

## APIs

* [OpenWeatherAPI](https://openweathermap.org/) - The API for thr weather information
* [Google Static Map API](https://developers.google.com/maps/documentation/maps-static/intro) - The API for the static map
